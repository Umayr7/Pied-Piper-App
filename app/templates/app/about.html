What is Huffman Algorithm ?

Huffman coding is a lossless data compression algorithm. In this algorithm, a variable-length code is assigned to input different characters. The code length is related to how frequently characters are used.. First one to create a Huffman tree, and another one to traverse the tree to find code. HUFFMAN ALGORITHM.



The process behind its scheme includes sorting numerical values from a set in order of their frequency.

How Compression works ?

Compression works in a sequence.

The algorithm first check the occurence(frequency) of each value and then sort all the values with respect to their frequency. After this, a tree is formed based on that sorted values.


Then each value gets an encoded data by traverse the value to the root with help of tree, and give value at each branch which is either 0 (if branch is left) or 1 (if branch is right). then write these encoded data in sequence of user given data. Then finally create a binary file of it.

How Decompression works ?

We decompress the user given compressed file by extracting key from it that we inserted. After that we create a tree with the help of our meta and decode the the values.
